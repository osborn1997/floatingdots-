HTML5

HyperText Markup Language also popularly known as HTML is one of the most important components of a website or a web aplication. It is often referd to as the skeletal part of the website because Html forms the backbone of every website regardless of what backend languages used for the web application. It is often and correctly argued and frouned upon by professional programers when it is called a programing language, but there is no doubt that this markup language could make a website stand without any other languages added to it.

Cascading style sheet also known as CSS is the second most important language in designing a website.Css gives that clasyy, custom and styleish look to the web abliction. No doubt Html can create and run a website , but unless your aim is to make a replica of craigslist then Css is definetly important to know.Html is very limited when it comes to design, Css gives the colors and shapes and positions to all the elements created by HTML.
